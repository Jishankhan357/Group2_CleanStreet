## 🎯 Main Features

### 1. Browse Issues ✅
- View all community-reported issues
- See issue title, description, and image
- View reporter info and date posted
- See category and status badges

### 2. Filter & Search ✅
- Filter by 9 categories
- Search by title, description, or address
- Pagination (20 issues per page)
- Reset filters easily

### 3. Like Issues ✅
- Like/unlike with one click
- See real-time like count
- Visual feedback (filled heart)
- Login required

### 4. Comment ✅
- Add comments to issues
- View all comments with timestamps
- See commenter's name and avatar
- Comments respect reporter settings

### 5. Manage Comments ✅
- Like/unlike comments
- Delete your own comments
- Confirmation dialog on delete
- Real-time updates

### 6. Share Issues ✅
- Share via WhatsApp
- Share via Facebook
- Share via Twitter
- Copy direct link to clipboard

### 7. View Details ✅
- Open issue in modal dialog
- See all images
- View full description
- See all comments and interactions

## 🏗️ Architecture

```
┌─────────────────────────────────────────────────┐
│              USER INTERFACE (React)             │
│  Community Page | Issue Card | Comment Section  │
└──────────────────┬──────────────────────────────┘
                   │
                   ↓ HTTP/JSON
┌─────────────────────────────────────────────────┐
│           API LAYER (Express.js)                │
│  GET /feed | POST /comment | POST /like | etc.  │
└──────────────────┬──────────────────────────────┘
                   │
                   ↓ Mongoose
┌─────────────────────────────────────────────────┐
│        DATABASE LAYER (MongoDB)                 │
│  Reports Collection with enhanced schema        │
└─────────────────────────────────────────────────┘
```

---

## 🔄 Data Flow

```
User Actions                    Backend                  Database
───────────────                 ───────                  ────────

View Issues ────GET /feed────→ Query reports ─→ MongoDB
                               with filters
                               
Like Issue ──POST /like──→ Update likes array ─→ MongoDB

Add Comment ─POST /comment─→ Add to comments ─→ MongoDB
                           array

Share ──POST /share──→ Increment shares ──→ MongoDB
```

### 1. No Database Migration Needed
```

```


```bash
cd frontend && npm start
```
